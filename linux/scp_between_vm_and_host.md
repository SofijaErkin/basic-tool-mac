# SCP Command Combat Style

## 1.Summary

SCP: Copy a File Between Two Systems

## 2.Usage

`scp [OPTION] [user@]SRC_HOST:]file1 [user@]DEST_HOST:]file2`

Notice:

In SCP command,

`-P` - Specifies the remote host ssh port.

`-p` - Preserves files modification and access times.

In SSH command,

`-p` - Specifies the remote host ssh port.
